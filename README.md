# Python图书推荐管理系统 python1234

本项目是一款基于Django框架的图书推荐管理系统，集成了数据可视化功能，旨在为用户提供便捷的图书信息管理及智能推荐。

## 技术栈

- **后端**: Python 3, Django
- **前端**: HTML, CSS, JavaScript
- **数据库**: PostgreSQL
- **数据可视化**: Matplotlib, Chart.js

## 功能模块

- **用户管理**: 用户注册、登录、个人信息管理等
- **图书管理**: 图书信息录入、编辑、删除等
- **推荐系统**: 根据用户喜好智能推荐图书
- **数据可视化**: 展示图书分类、用户活跃度等统计信息

### 系统角色

- **普通用户**: 可浏览图书、管理个人信息、查看推荐图书
- **管理员**: 拥有图书管理权限，可编辑、删除图书信息

## 运行环境

- **操作系统**: Linux / macOS / Windows
- **Python版本**: Python 3.6 或以上
- **依赖管理**: pip, virtualenv

## 部署步骤

1. 克隆项目代码到本地
2. 使用virtualenv创建虚拟环境
3. 安装项目依赖
4. 配置数据库
5. 运行迁移创建数据库表
6. 启动开发服务器

## 核心亮点

- **智能化推荐**: 利用算法为用户提供个性化的图书推荐
- **数据可视化**: 直观展示用户与图书数据，提升决策效率
